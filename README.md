 Student Score Analyzer

This is a simple C project which takes marks of students and finds the class average, highest and lowest marks.  
It helps to understand how data can be stored in arrays and processed using loops and functions in C.

C Concepts used:
- Arrays
- for and while loops
- Functions
- if else statements
- scanf and printf for input output

Math Concepts used:
- Mean (Average)
- Finding Maximum and Minimum values

How to Compile:
gcc main.c -o score_analyzer -lm

How to Run:
./score_analyzer
